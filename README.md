# SDN Loji 3 Kota Bogor - Next.js

Website resmi SDN Loji 3 Kota Bogor yang telah direfaktor menggunakan Next.js dan dioptimalkan untuk hosting di Vercel.

## Fitur Utama
- **Next.js App Router & TypeScript**: Struktur proyek modern berkinerja tinggi.
- **GSAP Animations**: Transisi loading, menu mobile, reveal effects, dan smooth scrolling menggunakan ScrollToPlugin.
- **Responsive & Premium Design**: Visual kelas premium yang disesuaikan dengan berbagai ukuran perangkat (desktop dan mobile).
- **SEO & Metadata Teroptimasi**: Lengkap dengan Open Graph, Twitter cards, dan JSON-LD Structured Data Schema untuk instansi sekolah.

## Cara Menjalankan Lokal
1. Jalankan perintah instalasi dependensi:
   ```bash
   npm install
   ```
2. Jalankan server pengembangan lokal:
   ```bash
   npm run dev
   ```
3. Bangun proyek untuk produksi:
   ```bash
   npm run build
   ```
